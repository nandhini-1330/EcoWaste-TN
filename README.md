# EcoWaste-TN
EcoWaste-TN is an AI-powered web application designed to improve waste segregation and disposal across Tamil Nadu. It helps citizens identify waste types instantly and guides them to the correct disposal methods, promoting a cleaner and more sustainable environment.

🎯 Objective
To build a smart and user-friendly platform that encourages proper waste segregation and supports sustainable waste management practices in Tamil Nadu.

🚀 Features

🌍 Multi-language Support (English & Tamil)

🤖 AI-Based Waste Classification

👤 User Authentication (Login & Register)

🏙️ Citizen & Admin Modules

📍 Smart Waste Disposal Guidance

💬 Integrated Chatbot Support

📊 Admin Dashboard for Monitoring

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Java Spring Boot

Database: MySQL

API Integration: AI/ML waste classification

💡 Future Enhancements

📱 Mobile App Integration

📸 Real-time Image Recognition

🗺️ Location-based Waste Collection Tracking

🔔 Smart Notifications & Alerts
